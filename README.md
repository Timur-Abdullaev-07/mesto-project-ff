## Установка и запуск
#### Для установки и запуска проекта необходимо:
Клонировать репозиторий

```
git clone git@github.com:Timur-Abdullaev-07/mesto-project-ff.git
```
Установить зависимости
```
npm install
```
Запустить проект
```
npm run dev
```
## Сборка

```
npm run build
```
